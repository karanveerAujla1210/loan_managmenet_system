# CRM UI Starter

Minimal React + TypeScript starter for the Enterprise CRM UI.

Quick start:

```bash
cd crm-ui-starter
npm install
npm run dev
```

Included:
- Theme tokens in `src/theme/tokens.ts` aligned to the design spec
- Sample components: `Button`, `Sidebar`
- Pages: `Login`, `Dashboard`

Next steps:
